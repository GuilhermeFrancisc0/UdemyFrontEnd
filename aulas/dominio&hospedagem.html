Domínio e Hospedagem

O que é um site?

É um diretório contendo vários arquivos com extensões variadas, como:

  • html
  • css
  • js
  • jpg
  • png
  • etc...

Hospedagem

O que é: publicar um site? por no ar? por em produção? fazer deploy? subir o site?

Pegar o diretório que está em sua máquina de desenvolvimento e colocar no servidor. Deixando disponível na internet.

Qual servidor escolher?

Depende:

  • Tecnologia
    • Estático(O servidor não precisa executar ações "complexas")
    • Dinâmico(O servidor precisa executar ações "complexas". Ex: login)
  • Número de acessos
    • Compartilhada(site pequeno, baixo custo, custo compartilhado, popular)
    • VPS("meio termo" entre compartilhada e dedicada)
    • Dedicado(máquina própria, grandes sites)
    • Cloud(Virtualização de várias máquinas físicas, pagar hora de uso, elasticidade, recurso extra)

Como fazer upload de arquivos?

Depende de qual serviço de hospedagem:

  • FTP(protocolo de transferência de arquivos)
  • SSH(acesso ao servidor atraves de linha de comando)
  • cPanel("acesso amigável",interface ilustrativa)

Cloud Computer

  • Azure(Microsoft)
  • AWS(Amazon)
  • Google plataform

Outras alternativas

  • github pages(versionamento de projetos)
  • Heroku(recurso temporario)
  • netfly
  • DigitalOcean

Domínio

O que é um domínio na web?

É o nome do seu negócio na internet,seja ele um site institucional, uma aplicação, um blog, um portal de notícias, etc.

O que é um subdomínio

É um nome secundário "pendurado" no domínio principal. Ex:

  • www.meusite.com.br
  • blog.meusite.com.br
  • app.meusite.com.br

Obs: Servidor DNS(servidor de nome e domínio),formado por colunas com nome e ip. Consultado primeiro para obter o número de ip referente ao nome informado.

Como comprar um domínio?

Depende:

  • .com?
  • .com.br?
  • .org?
  • .org.br?

Obs: Terminação .br, quem é responsável é o registro.br para obtenção do domínio.

E depois do domínio adquirido?

Basta apontar no nosso DNS o ip do servidor onde está loado no site.
