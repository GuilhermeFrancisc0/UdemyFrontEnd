Objetos

Objetos nativos são objetos que fazem parte da própria linguagem. Ex: string, number, boolean, array, object, date, math.

Sintaxe formal

Ex: var obj = new Object();

Sintaxe Literal

Ex: var obj = {};

Objetos do ambiente de hospedagem

Objetos acrescentados dependendo do ambiente que o código está sendo interpretado.

Ex: para browser o alert();

Construtores

Construtores são objetos que utilizam o operador 'new' para ter uma instância(ou "cópia") do objeto.
Ex:

Referência e Valor

Elementos primitivos(Number, String, Boolean) são passados como o valor, ou seja, caso sejam associados não mudam o valor um do outro.

Ex:
var n = 0;
var n1 = n;
n+1;
resultado: n = 1 e n1 = 0

Elementos não primitivos(array, object) são passados como referência, ou seja, caso sejam associados mudam o valor um do outro, pois apontam pro mesmo espaço de memória.

Ex:
var arr = [0];
var arr1 = arr;
arr1.push(3);
resultado: arr = [0,3] e arr1 = [0,3]
